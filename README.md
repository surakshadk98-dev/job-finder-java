# 💼 Job Finder API (Spring Boot)

A Java Spring Boot application that fetches and filters real-time job listings based on user skills.

## 🚀 Features
- Search jobs by skills
- Real-time API integration
- RESTful backend
- JSON response

## 🛠 Tech Stack
- Java
- Spring Boot
- REST API

## 🔗 API Endpoint
http://localhost:8080/jobs?skills=java

## 📌 Example Response
```json
[
  {
    "title": "Java Developer",
    "company": "XYZ",
    "location": "Remote",
    "link": "https://..."
  }
]
